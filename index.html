Prediction and Analysis of Spread of Covid-19 based on Socio-Economic factors

What is this project about

Introduction

The more civilized humans became, building cities and forging trade routes to connect with other cities, the more pandemics started to become likely.The last pandemic that was recorded before the covid 19 was the Influenza A virus.

The coronavirus being a virus infecting the respitory tract of an individual took a toll by infecting nearly 120M people all over the world. We have seen so far a very wavy graph that shows the spread of covid-19 in India itself. The first peak in India recorded on 18th September and then we saw the rise of cases again around February which lead to deadly second wave and the cases have again come to a visible low. We believe that there might be a large number of factors ranging from metrological and climatic factors to socio economic factors. Only considering socio economic factors we today propose a project that analyzes and predicts the spread of the virus.

The relevance of this project covers a large range of socio economic factors that might have affected the adversity of the COVID-19.

The process behind the project

Methodology

Data

Source of our Data

Our data has been sourced from ourworldindata.org/covid-deaths

Data Preprocessing

Data Cleaning & Imputation

Missing data imputation, removing duplicate rows, removing redundant columns, removing duplicate, highly correlated features and removing features which are less correlated to the target.

Outlier Removal

Outlier removal using isolation forest algorithm

Using GridSearchCV to loop through predefined hyperparameters. This will help in selecting the best parameters from the listed hyperparameters. Splitting the data for training and testing

Analysis

Correlation of Covid Cases with Socio-Economic Factors

To find out which Socio-Economic Factors have visible correlation, can be positive or negative.

Prediction

Testing various prediction models

We used the XGBoost, Light GBM, Support Vector Machine and Lasso Regression prediction models to see which prediction model will give the best accuracy.

Conclusion

The most accurate Model to use while predicting covid cases using Socio-Economic Factors is the XGBoost Model.


As Human Development Index rises the number of total cases decreases consistently.


As the number of hospital beds available increase, positivity rate tends to decrease which means that the number of new cases also decrease


As hand washing facilities increase the positive rate can be seen decreasing upto a point, which lets us assume that the number of new cases will decrease


It can be seen that as the GDP per capita increases, the total number of tests also increases
